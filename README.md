# Klavye Hız Testi

Türkçe kelimelerle hazırlanmış, tarayıcıda çalışan bir klavye hız testi uygulaması.

## Özellikler

- Rastgele Türkçe kelimeler üretir
- Doğru ve yanlış harfleri anlık olarak gösterir
- 30, 60, 120 ve 180 saniyelik test seçenekleri sunar
- WPM (dakikadaki kelime sayısı) hesaplar
- Doğruluk oranını gösterir
- Doğru yazılan kelime sayısını takip eder
- Uzun testlerde kelimeler otomatik olarak devam eder

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript
